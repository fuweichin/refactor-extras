# refactor-extras

Extra commands to refactor code for VS Code



## Usage

1. Move selection to a new file
   select some text in editor, right click, select "Move selection to a new file", choose a path for new file
2. Insert content from a file
   move caret to some position, right click, select "Insert content from a file", choose a file
3. Insert relative path
   move caret to some position, right click, select "Insert relative path", choose a file
